Chip-8 emulator I made as first Rust project
# CLONE
```bash
git clone https://github.com/sl4Jd/chip-8_emulator.git
```
```bash
cd chip-8_emulator
```
```bash
git submodule update --init --recursive 
```
# BUILD AND RUN
```bash
cargo run "path/to/rom"
```
Example
```bash
cargo run "chip8_roms/games/Pong (1 player).ch8"
```
